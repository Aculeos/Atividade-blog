HTML

HTML significa "Hypertext Markup Language" (Linguagem de Marcação de Hipertexto). É a linguagem padrão usada para criar e projetar páginas na web. Consiste em uma série de códigos que definem a estrutura, o conteúdo e os elementos de uma página da web, como texto, imagens, vídeos, links e outros recursos. HTML usa uma estrutura baseada em "tags" (etiquetas) que são inseridas no código para definir e organizar diferentes tipos de conteúdo. Cada tag tem uma função específica, como indicar cabeçalhos, parágrafos, links, listas, imagens, entre outros.
