CSS

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo usada para descrever a apresentação de documentos HTML (ou XML). Enquanto o HTML estrutura o conteúdo de uma página da web, o CSS determina como esse conteúdo é exibido, incluindo o layout, cores, fontes e outros aspectos visuais.
